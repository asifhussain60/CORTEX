# Reports Category

**Purpose:** Status reports, test results, validation reports, and session completion documentation.

**Auto-Detection Patterns:**
- `SESSION-*.md` - TDD session completion reports
- `*-report.md` - General reports
- `*test*results*.md` - Test execution results
- `TDD-SESSION-*.md` - TDD workflow reports
- `*validation*report*.md` - Validation reports
- `*completion*report*.md` - Completion reports
- `*status*report*.md` - Status reports
- `*performance*report*.md` - Performance reports
- `*deployment*report*.md` - Deployment reports

**Common Document Types:**
- Session completion reports (TDD Mastery)
- Test execution results
- Validation reports (DoR/DoD)
- Performance analysis reports
- Deployment reports
- Status updates
- Sprint completion reports

**Index:** See [INDEX.md](./INDEX.md) for complete listing with date grouping.

**Organization:** Documents are automatically organized by date. Use the INDEX.md file to browse reports chronologically.
