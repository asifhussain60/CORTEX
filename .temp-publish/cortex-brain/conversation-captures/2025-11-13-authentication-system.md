## How to Capture

1. **In Copilot Chat:** Select all conversation text (Ctrl+A in chat window)
2. **Copy:** Ctrl+C
3. **Paste here:** Replace this entire file content with your conversation
4. **Save:** Ctrl+S
5. **Next:** Say "import conversation" to CORTEX

---

## What to Include

✅ **Include:**
- All user questions/requests
- All CORTEX responses
- Challenge/Accept sections
- Design decisions
- File references
- Next Steps sections

❌ **Exclude:**
- VS Code UI elements
- Timestamps (unless critical)
- Unrelated conversations
