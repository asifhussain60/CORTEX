# CORTEX Extension Changelog

All notable changes to the CORTEX VS Code extension will be documented in this file.

## [1.0.0] - TBD (Target: 2026-02-01)

### Added
- Initial release
- @cortex chat participant
- Token dashboard with real-time metrics
- Automatic conversation capture
- Lifecycle management (focus/blur checkpoints)
- External conversation monitoring (GitHub Copilot)
- Proactive resume prompts
- ML-powered token optimization
- Commands: /resume, /checkpoint, /history, /optimize
- Configuration options for all features

### Infrastructure
- Python ↔ TypeScript HTTP bridge
- BrainBridge communication layer
- WorkingMemory integration (Tier 1)
- Token metrics integration (Phase 1.5)

## [0.1.0] - 2025-11-08

### Added
- Extension scaffold
- Project structure
- TypeScript configuration
- Python bridge server
- Core module interfaces
