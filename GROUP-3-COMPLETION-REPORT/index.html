    Integration Points¶

    Cross-Tier Integration¶

    Tier 1 ‚Üí Tier 2: - Conversation entities feed pattern learning - Intent recognition improves knowledge graph queries

    Tier 2 ‚Üí Tier 3: - Patterns inform velocity interpretation - Insights enhance pattern metadata

    Tier 3 ‚Üí Tier 1: - Context enriches conversation metadata - File hotspots highlight conversation topics

    All Tiers ‚Üí Tier 0: - Metrics inform governance rules - Insights trigger governance checks

    Agent Integration (Future - Group 4)¶

    IntentRouter: - Uses Tier 1 conversation history for context - Queries Tier 2 for similar past intents - Checks Tier 3 for project health before routing

    WorkPlanner: - Uses Tier 3 velocity for task estimation - Queries Tier 2 for workflow templates - Records plans in Tier 1

    ChangeGovernor: - Uses Tier 3 file hotspots for risk assessment - Queries Tier 2 for anti-patterns - Logs warnings in Tier 1
