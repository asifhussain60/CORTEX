# Conversation Captures Category

**Purpose:** Imported conversations, chat logs, and dialogue transcripts from GitHub Copilot or other sources.

**Auto-Detection Patterns:**
- `capture_*.md` - Captured conversations
- `conversation-*.md` - Conversation logs
- `chat-log-*.md` - Chat logs
- `*transcript*.md` - Transcripts
- `*dialogue*.md` - Dialogue captures

**Common Document Types:**
- GitHub Copilot conversation imports
- Important chat log exports
- Decision-making dialogues
- Brainstorming session captures
- Technical discussion transcripts
- Problem-solving conversations

**Index:** See [INDEX.md](./INDEX.md) for complete listing with date grouping.

**Organization:** Documents are automatically organized by date, preserving important conversations for future reference.

**Note:** Conversations are imported for reference and knowledge retention. They help CORTEX learn from past interactions and maintain context across sessions.
