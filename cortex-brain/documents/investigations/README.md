# Investigations Category

**Purpose:** Bug investigations, issue analysis, troubleshooting documentation, and debug logs.

**Auto-Detection Patterns:**
- `bug-*.md` - Bug investigation reports
- `issue-*.md` - Issue analysis
- `debug-*.md` - Debug logs
- `*troubleshoot*.md` - Troubleshooting guides
- `*investigation*.md` - Investigation reports
- `*root*cause*.md` - Root cause analysis
- `*incident*.md` - Incident reports

**Common Document Types:**
- Bug investigation reports
- Issue analysis
- Root cause analysis
- Debug session logs
- Troubleshooting documentation
- Incident post-mortems
- Error analysis

**Index:** See [INDEX.md](./INDEX.md) for complete listing with date grouping.

**Organization:** Documents are automatically organized by date and issue type, making it easy to track investigation history.
