Template Categories¶

Help & Status - help_table - help_detailed - quick_start - status_check

Planning - work_planner_success - planning_dor_complete - planning_dor_incomplete - planning_security_review

TDD & Testing - tdd_workflow_start - test_generation_triggers - refactor_triggers

Administration - system_alignment_report - cleanup_operation - design_sync_operation

General - fallback (default template)

Template Configuration¶

Templates are defined in cortex-brain/response-templates.yaml:

templates:
  help_table:
    triggers:
    - help
    - /help
    - what can cortex do
    response_type: table
    content: |
      [Template content]

Using Templates¶

Templates are auto-selected based on: 1. Exact trigger match (highest priority) 2. TDD workflow detection 3. Planning workflow detection 4. Fuzzy match (70%+ similarity) 5. Fallback (default)
