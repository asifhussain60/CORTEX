## Visual Style

### Color Palette
- **Background:** Dark slate gray (#2C3E50) with subtle technical grid pattern
- **Primary Text:** Pure white (#FFFFFF)
- **Section Headers:** Bright cyan (#00D4FF) - developer-friendly accent
- **Code Snippets:** Monospace font with syntax highlighting (green/blue/orange)
- **Icons:** Minimalist line icons in cyan/white
- **Borders:** Thin cyan lines (#00D4FF, 2px)

### Typography
- **Title:** Bold sans-serif, 48pt, white
- **Section Headers:** Bold sans-serif, 32pt, cyan
- **Body Text:** Sans-serif, 18pt, white
- **Code:** Monospace font, 16pt, syntax-highlighted

### Visual Elements
- Subtle grid pattern in background (developer aesthetic)
- Section dividers as clean horizontal lines
- Code blocks with dark background and syntax highlighting
- Icons using clean, minimal line style
- Professional technical documentation feel
- No decorative elements - pure functionality

### Composition
- Balanced two-panel layout (70/30 split)
- Clear visual hierarchy (title ‚Üí sections ‚Üí content)
- Breathing room between sections
- Code snippet prominently featured
- Reference card easily scannable
- Professional, authoritative design
