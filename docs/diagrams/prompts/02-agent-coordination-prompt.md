### Message Routing (Corpus Callosum → Agents)
- **Outbound Messages:** Solid arrows from router to each agent
- **Arrow Style:** Curved paths with gradient (yellow → hemisphere color)
- **Labels:** "REQUEST", "TASK", "QUERY" positioned on arrow paths
- **Width:** 4px arrows indicating message flow strength

### Result Feedback (Agents → Corpus Callosum)
- **Return Paths:** Dashed arrows from agents back to router
- **Arrow Style:** Curved paths with gradient (hemisphere color → yellow)
- **Labels:** "RESULT", "STATUS", "OUTPUT" positioned on return paths
