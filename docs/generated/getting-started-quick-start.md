## Prerequisites

- Python 3.8 or higher

- Git
- VS Code (recommended)

## Installation

### 1. Clone the Repository

```bash
git clone 
cd CORTEX
```

### 2. Install Python Dependencies

```bash
pip install -r requirements.txt
```

### 3. Install Node Dependencies (for VS Code extension)

```bash
cd cortex-extension
npm install
```

### 4. Configure CORTEX

Copy the example configuration:

```bash
cp cortex.config.example.json cortex.config.json
```

Edit `cortex.config.json` with your settings:

```json
{
  "brain_path": "",
  "tier0_protection": ,
  "auto_learning": 
}
```

## First Steps

python -m src.main
